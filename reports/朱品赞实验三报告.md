3. 运行程序验证结果。

**代码实现：**

```python
import time
import random

print("开始".center(25, '-'))
progress = 0
while progress < 100:
    # 随机增长1-5个百分点
    increment = random.randint(1, 5)
    progress += increment
    if progress > 100:
        progress = 100
    
    b='*'*progress
    c='.'*(100-progress)
    print("\r{0}%[{1}->{2}]".format(progress,b,c),end='')
    
    # 随机延时0.2-0.8秒
    sleep_time = random.uniform(0.2, 0.8)
    time.sleep(sleep_time)
print()
print("{:-^25}".format("结束"))
```

**运行结果：**
实现了随机增长的进度条，每次增长1-5个百分点，延时0.2-0.8秒，直到100%完成。

def replace_sensitive_words():
    sensitive_word = input("请输入敏感词：")
    sentence = input("请输入一句话：")
    
    # 将敏感词替换为*
    replaced_sentence = sentence.replace(sensitive_word, '*' * len(sensitive_word))

    print("替换后：", replaced_sentence)
